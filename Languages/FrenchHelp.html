Zen l'Initié - Aide

Déplacement des pions :

  • Dans le mode débutant : Pour sélectionner le pion que vous souhaitez déplacer, vous pouvez cliquer sur celui-ci ou bien saisir directement son numéro (ou bien 'z' dans le cas du Zen) dans la case située entre les flèches.
  • Dans le mode avancé : Pour sélectionner le pion que vous souhaitez déplacer vous pouvez cliquer sur celui-ci ou bien rentrer son numéro (ou bien 'z' pour le zen) dans la case "pion". Pour indiquer la case sur laquelle vous souhaitez le déplacer, vous pouvez cliquer sur celle-ci ou bien rentrer le numéro de la ligne et la lettre correspondant à la colonne dans les cases "ligne" et "colonne".

Principales commandes et fonctionnalités :

  • Pour revenir au menu, cliquez sur Menu -> Retour au menu
  • Pour changer la langue du jeu, cliquez sur Paramètres -> Langue puis choisissez votre langue.
  • Pour couper ou réactiver le son de l'application, cliquez sur Paramètres -> Son.
  • Pour changer le thème de l'application, cliquez sur Paramètres -> Thème et sélectionnez le thème que vous préférez.
  • Pour voir les règles du jeu, cliquez sur l'icône ? -> Règles.
  • Pour sauvegarder votre partie, cliquez sur Partie -> Sauvegarder, si votre partie a déjà été sauvegarder dans un fichier, ce fichier sera actualisé, sinon une fenêtre vous demandera où vous souhaitez enregistrer votre fichier. Si vous souhaitez enregistrer votre fichier dans un nouvel emplacement, cliquez sur "enregistrer-sous". Si un message d'erreur s'affiche, cela peut être dû au fait que le fichier dans lequel vous souhaitez enregistrer votre partie est ouvert dans une autre application. Si c'est le cas, fermez le fichier et recommencez l'enregistrement (attention, enregistrer votre partie supprimera le contenu du fichier), si ce n'est pas le cas, vérifiez le chemin d'accès renseigné.
