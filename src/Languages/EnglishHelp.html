Zen l'Initié - Help

Pawns' displacements :

  • In the beginner's mode : To select the pawn you wish to move, you can click on it or enter its number (or 'z' in the case of Zen) in the box between the arrows.
  • In advanced mode: To select the pawn you want to move you can click on it or enter his number (or 'z' for Zen) in the "pawn" square. To indicate the square on which you wish to move it, you can click on it or enter the number of the line and the letter corresponding to the column in the "row" and "column" boxes.

Main commands and features :

  • To return to the menu, click Menu -> Back to menu
  • To change the language of the game, click Settings -> Language and choose your language.
  • To mute or unmute the sound of the application, click on Settings -> Sound.
  • To change the theme of the application, click Settings -> Theme and select the theme you prefer.
  • To see the rules of the game, click on the icon ? -> Rules.
  • To save your game, click on game -> Save, if your game has already been saved to a file, this file will be updated, otherwise a window will ask you where you want to save your file. If you want to save your file in a new location, click on "save-under". If an error message is displayed, this will may be due to the fact that the file in which you want to save your game is open in another application. If this is the case, close the file and start the recording again (be careful, saving your game will delete the contents of the file), if not, check the path you entered.
